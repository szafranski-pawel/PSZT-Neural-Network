# PSZT - Project "Defaults 2"

## Requirements

```bash
pip3 install -r requirements.txt
```


## Usage

Example:

```sh
./skynet.py
```

Optional arguments: 

| Parameter                 | Default       | Description   |	
| :------------------------ |:-------------:| :-------------|
| --file\_path	            |	dane.csv  | Path to data set |
| --learn\_rate             |   0.1       | Learn rate |
| --neurons	                |	[]        | Neurons configuration |
| --epochs 		            |   100       | Number of epochs used in training |
| --batch\_size 		    |   5         | How often to recalculate gradient|
